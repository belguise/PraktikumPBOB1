## Deskripsi Umum

Repositori ini berisi kumpulan hasil praktikum mata kuliah **Pemrograman Berorientasi Objek (PBO)** yang dikerjakan menggunakan bahasa pemrograman Java.

Praktikum ini bertujuan untuk memahami konsep dasar hingga lanjutan dalam paradigma **Object-Oriented Programming (OOP)**, seperti:

* Class dan Object
* Encapsulation
* Inheritance
* Polymorphism
* Overloading dan Overriding

Setiap pertemuan praktikum berisi implementasi konsep-konsep tersebut dalam bentuk studi kasus dan aplikasi sederhana.

---

## Tujuan Praktikum

1. Memahami konsep dasar pemrograman berorientasi objek
2. Mampu mengimplementasikan class dan object dalam Java
3. Memahami penggunaan konstruktor, method, dan atribut
4. Menerapkan konsep inheritance (pewarisan)
5. Memahami dan mengimplementasikan polymorphism
6. Mengembangkan kemampuan analisis dan pemecahan masalah berbasis OOP

## Cara Menjalankan Program

1. Buka terminal atau command prompt
2. Masuk ke folder praktikum
3. Compile semua file:

```
javac *.java
```

4. Jalankan program:

```
java NamaFileMain
```

Contoh:

```
java AplikasiMahasiswa
java AplikasiSeminar
```
